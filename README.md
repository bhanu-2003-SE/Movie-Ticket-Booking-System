Technology Stack: Built using C# and .NET Framework for Windows desktop environments.

UI Components: Interactive Windows Forms (WinForms) for seamless user and admin workflows.

Serverless Storage: Utilizes XML files for robust data storage, handling movie schedules, seat availability, and user bookings without requiring SQL servers.

Core Functionalities: Full CRUD operations for ticket booking, real-time ticket availability updates, and basic user/admin roles.
